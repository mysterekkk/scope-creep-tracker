# Changelog

## [1.0.0] - 2026-03-23

### Added

- Project management with budget tracking (hours or fixed price)
- Task management with automatic scope flag detection
- Built-in time tracking (start/stop timer + manual entries)
- Client message logger with heuristic scope creep detection
- Scope creep detection dashboard with visual metrics
- PDF change request generation via @react-pdf/renderer
- Client-ready message template with copy-to-clipboard
- Multi-format export: PDF, CSV, JSON
- Dark/light mode with system preference detection
- Multi-language support: English, Polish, Spanish, German, French
- 100% client-side processing (privacy-first, localStorage)
